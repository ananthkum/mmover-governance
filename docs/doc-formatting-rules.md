# Documentation Formatting Rules — Single-line 140

Rules:
- Max 140 characters per line.
- Headings single-line.
- Bullets single-line; if too long, split into multiple bullets.
- Avoid nested bullets beyond 1 level.
- Avoid tables where possible; never embed code blocks in tables.
- Code blocks allowed; each code line must also be ≤140.
- Use clear sectioning and client-action banners where needed.
