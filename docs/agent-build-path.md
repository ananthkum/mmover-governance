# Agent Build Path — Decision Rule

Use Agent Builder when:
- The agent is declarative.
- The agent mainly needs instructions, starter prompts, SharePoint or public website knowledge, and simple built-in capabilities.

Use Copilot Studio when:
- The agent needs actions to integrate external services.
- The agent needs more advanced orchestration, automation, or controlled publishing workflows.
- The agent needs richer management beyond the Agent Builder subset.

Escalation rule:
- Start declarative where possible.
- Escalate to Copilot Studio only when a documented requirement cannot be met in Agent Builder.
