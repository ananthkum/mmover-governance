# Agent Publish Checklist

Before publish:
- Name, description, instructions, and starter prompts finalized.
- Knowledge sources documented and approved.
- Billing model confirmed.
- Test prompts passed for positive, negative, edge, and abuse cases.
- Sharing scope approved.
- Admin controls verified.
- Version and changelog updated.

After publish:
- Smoke test in Microsoft 365 Copilot.
- Confirm discoverability and access scope.
- Record agent ID, publish date, owner, and rollback plan.
