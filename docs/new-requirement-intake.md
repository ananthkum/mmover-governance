# New Requirement Intake

Auto-detect:
- You do not need a prefix.
- Related to Active REQ updates the current REQ.
- Unrelated becomes a new requirement event and runs requirement matching.

Overrides:
- KEEP_ACTIVE REQ-###.
- SWITCH_ACTIVE REQ-###.

Mandatory first response content:
- REQ Dashboard.
- Translation brief when user input is layman or ambiguous.
- Candidate REQ match report.
- Impact analysis gate status.
- Client action banner.
