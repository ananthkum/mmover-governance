# Agent Instructions Policy

Instructions must include:
- Role and business purpose.
- In-scope topics.
- Out-of-scope topics and refusal behavior.
- Source priority order.
- Required output style.
- Hallucination guardrails.
- Escalation or clarification rule when confidence is low.
- Data handling limitations.

Instructions must not:
- Promise unsupported actions.
- Claim access to sources not configured.
- Expose hidden governance, secrets, or tenant-only implementation details.
